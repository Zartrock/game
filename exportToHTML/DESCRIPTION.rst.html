1    McCabe complexity checker 
2    ========================= 
3    
4    Ned's script to check McCabe complexity.
5    
6    This module provides a plugin for ``flake8``, the Python code checker.
7    
8    
9    Installation 
10   ------------ 
11   
12   You can install, upgrade, uninstall ``mccabe`` with these commands:: 
13    
14     $ pip install mccabe 
15     $ pip install --upgrade mccabe 
16     $ pip uninstall mccabe 
17    
18    
19   Standalone script 
20   ----------------- 
21   
22   The complexity checker can be used directly:: 
23    
24     $ python -m mccabe --min 5 mccabe.py 
25     ("185:1: 'PathGraphingAstVisitor.visitIf'", 5) 
26     ("71:1: 'PathGraph.to_dot'", 5) 
27     ("245:1: 'McCabeChecker.run'", 5) 
28     ("283:1: 'main'", 7) 
29     ("203:1: 'PathGraphingAstVisitor.visitTryExcept'", 5) 
30     ("257:1: 'get_code_complexity'", 5) 
31    
32    
33   Plugin for Flake8 
34   ----------------- 
35   
36   When both ``flake8 2.0`` and ``mccabe`` are installed, the plugin is
37   available in ``flake8``:: 
38    
39     $ flake8 --version 
40     2.0 (pep8: 1.4.2, pyflakes: 0.6.1, mccabe: 0.2) 
41    
42   By default the plugin is disabled.  Use the ``--max-complexity`` switch to
43   enable it.  It will emit a warning if the McCabe complexity of a function is
44   higher that the value:: 
45    
46       $ flake8 --max-complexity 10 coolproject 
47       ... 
48       coolproject/mod.py:1204:1: C901 'CoolFactory.prepare' is too complex (14) 
49    
50   This feature is quite useful to detect over-complex code.  According to McCabe,
51   anything that goes beyond 10 is too complex.
