32   ========
33   Overview
34   ========
35   
36   
37   
38   A fast and thorough lazy object proxy.
39   
40   * Free software: BSD 2-Clause License
41   
42   Note that this is based on `wrapt`_'s ObjectProxy with one big change: it calls a function the first time the proxy object is
43   used, while `wrapt.ObjectProxy` just forwards the method calls to the target object.
44   
45   In other words, you use `lazy-object-proxy` when you only have the object way later and you use `wrapt.ObjectProxy` when you
46   want to override few methods (by subclassing) and forward everything else to the target object.
47   
48   Example::
49   
50       import lazy_object_proxy
51   
52       def expensive_func():
53           from time import sleep
54           print('starting calculation')
55           # just as example for a very slow computation
56           sleep(2)
57           print('finished calculation')
58           # return the result of the calculation
59           return 10
60   
61       obj = lazy_object_proxy.Proxy(expensive_func)
62       # function is called only when object is actually used
63       print(obj)  # now expensive_func is called
64   
65       print(obj)  # the result without calling the expensive_func
